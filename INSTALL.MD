# Basic Install Guide
1. Install ox_lib if it isn't already installed https://github.com/CommunityOx/ox_lib
2. Install PolyZone if it isn't already installed https://github.com/mkafrin/PolyZone/releases/latest
3. Install motion_bridge if it isn't already installed https://github.com/Motion-Scripts/motion_bridge
4. Download latest release.
5. Add script to your Fivem resources folder.
6. Add `ensure motion-redzones` to `server.cfg`
7. Start to config the script
- Add any zones you need in shared/config.lua
- Setup webhooks in shared/server_config.lua
- Setup kill tracking settings in shared/server_config.lua

# Dependencies
motion_bridge (for ambulance and inventory)
ox_lib
PolyZone
